= 更新 vCenter Server 和 ONTAP 認證
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用維護主控台更新 vCenter Server 執行個體和 ONTAP 認證。

*您需要的是什麼*

您必須擁有維護使用者登入認證。

*關於此工作*

如果您已變更 vCenter Server 、 ONTAP 或 Data LIF 部署後的認證、則需要使用此程序更新認證。

*步驟*

. 從 vCenter Server 開啟 VASA Provider 的主控台。
. 以維護使用者身分登入。
. 輸入 `2` 以選取系統組態功能表。
. 輸入 `9` 以變更 ONTAP 認證。
. 輸入 `10` 以變更 vCenter 認證。
